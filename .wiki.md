# Project Summary

律审雷达是一个法律文书智能审查与生成平台，面向合同审查、案件材料整理、法律文书生成和深度法律风险报告。当前项目由 FastAPI 后端、React/Vite 前端、本地 SQLite 数据库、法律知识库 seed 和多阶段法律分析流程组成。

# Current Architecture

- **Frontend**: React 18、TypeScript、Vite、Tailwind CSS、shadcn/ui 风格组件。
- **Backend**: FastAPI、SQLAlchemy asyncio、本地 SQLite 开发库，可通过环境变量切换数据库。
- **AI Gateway**: OpenAI-compatible 网关，由 `APP_AI_BASE_URL` 和 `APP_AI_KEY` 配置。
- **Document Pipeline**: 文本层解析优先，OCR 兜底，面向 PDF/Word/文本上传。
- **Legal Review Pipeline**: 文档类型识别、规则预扫描、模型分析、法律依据检索、质量审计、报告组装。
- **Legal Knowledge**: `legal_knowledge` 本地知识库，当前覆盖合同审查高频《民法典》法条。

# Main Modules

- **Authentication**: 用户注册、登录、会话和管理员初始化。
- **Document Management**: 文档上传、解析、存储和案件材料关联。
- **Deep Legal Review**: 多阶段法律审查、风险识别、替代条款、法律依据引用和专业报告生成。
- **Document Strategy**: 针对合同、租赁、劳动、买卖、借款、保证、股权转让、诉讼文书等类型的专门审查规则。
- **Legal Knowledge**: 本地 SQLite/JSON seed、更新脚本和检索接口。
- **Case Workspace**: 案件、事实、当事人、材料和任务管理。
- **Legal Draft Generation**: 起诉状、答辩状、律师函、仲裁申请等文书生成。
- **Payment**: Stripe 支付相关服务。

# Important Files

- `README.md`: 项目入口和本地运行命令。
- `docs/PROJECT_STRUCTURE.md`: 当前项目结构说明和扩展约定。

- `app/frontend/src/pages/DeepReportPage.tsx`: 深度审查报告页面。
- `app/frontend/src/lib/deepReviewApi.ts`: 深度审查 API 客户端。
